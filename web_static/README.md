# Web Static Project

This project is part of the AirBnB clone project and focuses on the development of the web static part of the application.

## Table of Contents

- [Description](#description)


## Description

The web_static project is responsible for creating the static HTML and CSS files that will be served by the web server. It includes the following features:

- HTML templates: The project includes HTML templates that are used to generate the final HTML files dynamically.
- CSS stylesheets: The project includes CSS stylesheets that define the visual appearance of the web pages.
- Static assets: The project includes static assets such as images, fonts, and JavaScript files that are used by the web pages.
